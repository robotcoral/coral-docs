# Editor
This chapter covers the editor features.

## Codemirror 6

The editor itself works using [Codemirror 6](https://codemirror.net/6/).

### Features
Editor features include:

- Syntax highlighting
- Autocompletition
- Automtatic bracket closing
- Undo/Redo history
  ... and much more
